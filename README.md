# README #

My (previous) personal blog and code playground.

# Install #

1. clone the repo.
2. cd into the repo directory.
3. run npm install
4. run bower install
5. download and extract Wordpress into the [repo_dir]/wordpress
6. install WAMP/MAMP/whatever
7. run wordpress install

# Wordpress Plugins #

* TinyMCE Advanced
* WP REST API
* Media Tags
